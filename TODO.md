# Дорожная карта проекта (Roadmap)

## 🎯 Основная цель: От дрона к трактору
Создать Open Source инструмент, который позволяет фермеру загрузить карту с дрона (Mavic 3M), увидеть проблемные зоны и отправить карту задания в трактор.

## ✅ Выполнено (Foundation)
- [x] Базовая архитектура (SPA + REST API).
- [x] Управление границами полей (рисование, импорт Shapefile).
- [x] Учет земельного банка (собственность/аренда).
- [x] Мобильная версия (PWA) для работы в поле.
- [x] **Точность площадей:** Исправлен расчет гектаров (EPSG:3035).
- [x] **Экспорт DJI KMZ (WPML 1.0.6):**
    - [x] Полная совместимость с Mavic 3M (проверено на пульте).
    - [x] Настройка перекрытий, высоты и угла курса (Course Angle).
    - [x] Человекочитаемые названия файлов.
    - [x] Понятные UI-подсказки для агронома.
- [x] **Детали поля:** Отдельная страница с мини-картой и полной информацией.
- [x] **Работа с NDVI (GeoTIFF):** Загрузка до 1ГБ, зонирование (K-Means), отрисовка.
- [x] **Исправление ошибок:** Решена проблема с CORS для DataTables (перевод встроен локально).
