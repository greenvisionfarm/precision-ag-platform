# Field Mapper

Field Mapper — это веб-приложение для визуализации и управления географическими данными полей. Оно позволяет пользователям загружать shape-файлы в ZIP-архивах, отображать их на интерактивной карте и просматривать в виде таблицы.

## Основные возможности

- **Загрузка Shape-файлов:** Загрузка полей из ZIP-архива, содержащего shape-файл.
- **Интерактивная карта:** Отображение загруженных полей на карте с использованием Leaflet.
- **Визуализация данных:** Каждое поле можно нажать для просмотра основной информации.
- **Список полей:** Отдельная страница с таблицей всех загруженных полей, их площади и других атрибутов.
- **Удаление полей:** Возможность удалять поля как на карте, так и в таблице.
- **Переименование полей:** Возможность переименовывать поля как на карте, так и в таблице.
- **Гибкое определение имени поля:** При загрузке имя поля определяется по приоритету из различных атрибутов shape-файла (например, 'Field_Name', 'name', 'NAME', 'ID').
- **Расчет площади:** Автоматический расчет площади для каждого полигона.

## Стек технологий

- **Backend:**
  - **Веб-фрейморк:** [Tornado](https://tornadoweb.org/en/stable/) (асинхронный)
  - **Гео-обработка:** [GeoPandas](https://geopandas.org/)
  - **ORM:** [Peewee](http://docs.peewee-orm.com/)
  - **База данных:** SQLite
  - **Тестирование:**
    - **Фреймворк:** Pytest
    - **Асинхронные тесты:** pytest-asyncio
- **Frontend:**
  - **Картография:** [Leaflet.js](https://leafletjs.com/)
  - **Плагин для рисования:** [Leaflet.draw](https://github.com/Leaflet/Leaflet.draw)
  - **Таблицы:** [DataTables](https://datatables.net/)
  - **AJAX и DOM:** [jQuery](https://jquery.com/)

## Установка и запуск

1.  **Клонируйте репозиторий:**
    ```bash
    git clone <your-repository-url>
    cd field_mapper
    ```

2.  **Создайте и активируйте виртуальное окружение:**
    ```bash
    python -m venv venv
    source venv/bin/activate  # Для Linux/macOS
    # venv\Scripts\activate    # Для Windows
    ```

3.  **Установите зависимости:**
    ```bash
    pip install -r requirements.txt
    ```

4.  **Запустите приложение:**
    ```bash
    python app.py
    ```

5.  **Откройте приложение в браузере:**
    Перейдите по адресу [http://localhost:8888](http://localhost:8888).

## API Эндпоинты

- `GET /`: Главная страница с картой.
- `GET /fields_list`: Страница с таблицей полей.
- `POST /upload`: Эндпоинт для загрузки ZIP-архива с shape-файлом.
- `GET /api/fields`: Возвращает все поля в формате GeoJSON для отображения на карте.
- `GET /api/fields_data`: Возвращает данные для заполнения таблицы DataTables.
- `DELETE /api/field/delete/<id>`: Удаляет поле по его уникальному идентификатору.
- `PUT /api/field/rename/<id>`: Переименовывает поле по его ID.
