# Управление полями

## Создание поля

### Способ 1: Рисование на карте

1. Откройте вкладку **🗺️ Карта**
2. Нажмите кнопку рисования на панели инструментов
3. Выберите инструмент **Polygon**
4. Нарисуйте границу поля кликами по карте
5. Дважды кликните для завершения
6. Введите название и сохраните

### Способ 2: Импорт Shapefile

1. Откройте вкладку **📥 Загрузки**
2. Выберите ZIP-архив с Shapefile (.shp, .shx, .dbf, .prj)
3. Нажмите **Загрузить**
4. Дождитесь обработки — поле появится на карте

### Способ 3: Из кадастровой карты

> Требуется подключение к WMS сервису

1. Откройте **🗺️ Карта**
2. Включите слой кадастра
3. Кликните на участок
4. Нажмите **Импортировать контур**

---

## Редактирование поля

### Изменение границ

1. Откройте **📋 Список полей**
2. Найдите нужное поле
3. Кликните на **Редактировать**
4. Перетащите вершины на карте
5. Сохраните изменения

### Обновление информации

1. Откройте детальную view поля
2. Нажмите **Редактировать**
3. Измените:
   - Название
   - Владельца
   - Статус (Собственность/Аренда)
   - Кадастровый номер
4. Сохраните

---

## Удаление поля

⚠️ **Внимание:** Удаление необратимо!

1. Откройте **📋 Список полей**
2. Найдите поле
3. Нажмите **Удалить** в действиях
4. Подтвердите удаление

---

## Экспорт данных

### Экспорт в KMZ (для DJI Mavic 3M)

1. Откройте **📋 Список полей**
2. Нажмите **Экспортировать всё (.ZIP)**
   - Или экспортируйте отдельное поле из детальной view
3. Настройте параметры:
   - Высота полёта (м)
   - Направление (градусы)
   - Перекрытие по горизонтали (%)
   - Перекрытие по вертикали (%)
4. Скачайте ZIP-архив с `.kmz` файлом

## Статистика по полям

Откройте вкладку **📊 Статистика** для просмотра:

- Общее количество полей
- Общая площадь земельного банка
- Распределение по владельцам

## Советы

### 📏 Точность границ

- Используйте спутниковые снимки высокой точности
- Проверяйте границы по кадастровой карте
- Избегайте самопересечений полигонов

### 📁 Организация

- Группируйте поля по владельцам
- Используйте статусы для учёта аренды
- Заполняйте кадастровые номера

### 🚁 Подготовка к полёту

- Проверяйте площадь поля (оптимально 20-100 га)
- Экспортируйте KMZ за день до полёта
- Проверяйте настройки перекрытий в DJI Pilot
