# Экспорт KMZ для DJI Mavic 3M

KMZ (KML Zip) — формат полётных заданий для дронов DJI, совместимый с Mavic 3M и DJI Pilot.

## Формат WPML 1.0.6

Field Mapper экспортирует KMZ в формате **Waypoint Mission v1.0.6**:

- ✅ Совместимость с DJI Pilot 1.x и 2.x
- ✅ Углы курса (course angle) для съёмки
- ✅ Настройки перекрытий (overlap)
- ✅ Главная точка (home point)

---

## Пошаговый экспорт

### Экспорт одного поля

1. Откройте **📋 Список полей**
2. Кликните на нужное поле
3. В детальной view нажмите **Экспорт DJI KMZ**
4. Настройте параметры (см. ниже)
5. Скачайте `.kmz` файл

### Массовый экспорт

1. Откройте **📋 Список полей**
2. Нажмите **Экспортировать всё (.ZIP)**
3. Скачайте ZIP-архив с KMZ всех полей

## Параметры полёта

### Высота (м)

| Высота | GSD* | Площадь за пролёт | Время полёта |
|--------|------|-------------------|--------------|
| 80 м | 2.2 см/пикс | Узкая | Дольше |
| 100 м | 2.8 см/пикс | Средняя | Среднее |
| 120 м | 3.3 см/пикс | Широкая | Быстрее |

*GSD (Ground Sampling Distance) — размер пикселя на местности

**Рекомендация:** 100-120 м для NDVI, 80-100 м для детальной съёмки

### Направление (градусы)

Угол пролётов относительно севера:

- **0°/180°:** С севера на юг
- **90°/270°:** С запада на восток
- **45°:** Диагональные пролёты

**Рекомендация:** Перпендикулярно длинной стороне поля

### Перекрытие по горизонтали (%)

Перекрытие между соседними пролётами:

- **60%:** Стандарт для ортофото
- **70%:** Для 3D моделей
- **80%:** Для точных измерений

**Рекомендация:** 70% для NDVI

### Перекрытие по вертикали (%)

Перекрытие между кадрами в одном пролёте:

- **60%:** Стандарт
- **70%:** Для высокой точности

**Рекомендация:** 65-70%

## Использование в DJI Pilot

### Импорт задания

1. Скопируйте `.kmz` на пульт DJI Smart Controller
2. Откройте **DJI Pilot**
3. Перейдите в **Mission → Waypoint**
4. Нажмите **Import** и выберите файл
5. Проверьте параметры на карте

### Проверка перед полётом

- [ ] Поле корректно отображается на карте
- [ ] Высота безопасна для препятствий
- [ ] Home point установлен верно
- [ ] Батареи заряжены на 100%
- [ ] GPS сигнал сильный (>10 спутников)

### Запуск миссии

1. Нажмите **Start Mission**
2. Дрон взлетит и перейдёт к первой точке
3. Следите за телеметрией
4. При необходимости нажмите **Pause** или **Return to Home**

## Шаблон template.kml

Field Mapper генерирует `template.kml` для импорта в DJI Pilot:

```xml
<?xml version="1.0" encoding="utf-8"?>
<kml xmlns="http://www.opengis.net/kml/2.2">
  <Document>
    <name>Field Name</name>
    <Placemark>
      <name>Waypoint Mission</name>
      <ExtendedData>
        <Data name="missionType"><value>1</value></Data>
        <Data name="altitude"><value>100</value></Data>
        <Data name="course"><value>90</value></Data>
        ...
      </ExtendedData>
    </Placemark>
  </Document>
</kml>
```

## Советы

### 🔋 Планирование

- Разбивайте большие поля (>50 га) на несколько миссий
- Проверяйте прогноз погоды (ветер < 10 м/с)
- Рассчитывайте время полёта с запасом 20%

### 📱 Настройки камеры

Для Mavic 3M:
- **Режим:** Multispectral
- **Формат:** TIFF + JPEG
- **Интервал:** 2-3 секунды

### ⚠️ Безопасность

- Проверяйте зону полёта на препятствия
- Установите RTH (Return to Home) высоту выше препятствий
- Держите дрон в прямой видимости
