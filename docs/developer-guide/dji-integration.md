# Интеграция с DJI Mavic 3M

Field Mapper оптимизирован для работы с мультиспектральными данными дронов DJI (особенно Mavic 3M и Phantom 4 Multispectral).

## 🚀 Быстрый режим (Fast Grid Mode)

В отличие от классических GIS систем, Field Mapper не требует создания ортофотоплана (Stitching) для базового анализа. 

### Как это работает:
1. Вы загружаете ZIP-архив с "сырыми" снимками.
2. Система извлекает метаданные из XMP (тег 700) и EXIF.
3. Координаты GPS используются для привязки каждого снимка к полю.
4. Значения пикселей нормализуются для получения коэффициента отражения (Reflectance).
5. На основе сетки снимков строится агрегированная карта индексов.

## 📊 Нормализация данных

Для получения корректных значений NDVI/NDRE используется формула:
`Reflectance = (DN - BlackLevel) / (Exposure * Gain)`

Где:
- **DN**: Значение пикселя (Digital Number).
- **BlackLevel**: Уровень черного сенсора.
- **Exposure**: Выдержка.
- **Gain**: Усиление сенсора.

## 📂 Группировка файлов

Система автоматически распознает каналы DJI по суффиксам:
- `_NIR.TIF` — Ближний инфракрасный
- `_R.TIF` / `_RED.TIF` — Красный
- `_RE.TIF` — Крайний красный (Red Edge)
- `_G.TIF` — Зеленый

## 📲 Экспорт в DJI Pilot

Экспортируемые KMZ файлы совместимы с **DJI Pilot 2** (WPML формат). Они включают:
- Границы поля.
- Точки маршрута для сканирования или зоны для внесения.
- Стилизацию зон по цветам.
