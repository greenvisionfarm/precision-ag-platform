# AI Contexts для Field Mapper

Эта папка содержит контекстные файлы для AI ассистентов.

## 📁 Файлы

| Файл | Для кого | Описание |
|------|----------|----------|
| `qwen.md` | Qwen Code | Полный контекст проекта для Qwen |
| `gemini.md` | Google Gemini | Контекст для Gemini |
| `README.md` | — | Этот файл |

## 🤖 Как использовать

### Qwen Code
Файл `.ai-contexts/qwen.md` автоматически подхватывается Qwen Code при работе с проектом.
Содержит:
- Архитектуру проекта
- Стек технологий
- Git workflow (ветки, remote, Conventional Commits)
- Команды для разработки
- Структуру проекта
- Ключевой функционал
- Метрики
- Недавние изменения
- Подсказки для AI агентов

### Google Gemini
Откройте `.ai-contexts/gemini.md` или используйте команду:
```
/ctx .ai-contexts/gemini.md
```

## 📝 Содержание контекстов

Оба файла содержат:
- **Общую информацию:** название, GitHub repo, лицензия, статус
- **Архитектуру:** Docker Compose, стек технологий
- **Git Workflow:** ветки, remote, Conventional Commits, CI/CD
- **Команды:** запуск, тесты, Docker
- **Структуру проекта:** все директории и файлы
- **Ключевой функционал:** земельный учет, NDVI, экспорт, PWA
- **Метрики:** тесты, покрытие, размер образа
- **Roadmap:** milestones с датами
- **Недавние изменения:** лог обновлений проекта

## 🔄 Обновление

При значительных изменениях в проекте обновляйте оба файла:
1. Добавьте новые разделы в структуру
2. Обновите метрики
3. Добавьте записи в "Недавние изменения"
4. Обновите Git workflow если изменился процесс

## 🧠 Что знают AI агенты из этих файлов

### О Git
- Ветка `master` — стабильный код
- Remote `upstream` → GitHub репозиторий
- Conventional Commits для сообщений
- CI запускается автоматически при push
- GitHub Pages деплоится из `master`

### О разработке
- Type hints обязательны для Python
- ES6 модули для JavaScript
- Декоратор `@db_connection` для handler'ов
- Command pattern для обновлений полей
- Тесты обязательны для новой функциональности

### О проекте
- Платформа в активной разработке
- Не полностью протестирована в production
