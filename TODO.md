# Roadmap проекта Field Mapper

## ✅ Выполнено (2024-2026)

### Infrastructure
- [x] **Docker v2:** Образ с GDAL, Python 3.12, Node.js 20
- [x] **Фоновые задачи:** Huey + Redis для NDVI анализа
- [x] **Тестирование:** Pytest + Jest, интеграция в CI
- [x] **Рефакторинг 2026:** 
  - Backend: декораторы, валидация, type hints, command pattern
  - Frontend: ES6 модули, класс FieldMapperApp
  - Docker: оптимизация сборки (кэш, многоэтапность)

### Функциональность
- [x] **Земельный учет:** Границы полей, владельцы, кадастр
- [x] **NDVI анализ:** Загрузка GeoTIFF, зонирование
- [x] **DJI KMZ:** Экспорт WPML 1.0.6 (Mavic 3M)
- [x] **PWA:** Offline режим, service worker
- [x] **UI 2026:** Страница загрузок, улучшенная кнопка меню
