# AI Contexts для Field Mapper

Эта папка содержит контекстные файлы для AI ассистентов.

## 📁 Файлы

| Файл | Для кого | Описание |
|------|----------|----------|
| `qwen.md` | Qwen Code | Полный контекст проекта для Qwen |
| `gemini.md` | Google Gemini | Контекст для Gemini |

## 🤖 Как использовать

### Qwen Code
Откройте `.ai-contexts/qwen.md` или добавьте в `.qwen/output-language.md`:
```markdown
<!-- qwen-code:context: .ai-contexts/qwen.md -->
```

### Google Gemini
Откройте `.ai-contexts/gemini.md` или используйте команду:
```
/ctx .ai-contexts/gemini.md
```

## 📝 Содержание

Оба файла содержат:
- Архитектуру проекта
- Стек технологий
- Команды для разработки
- Структуру проекта
- Ключевой функционал
- Метрики
- Недавние изменения

## 🔄 Обновление

При значительных изменениях в проекте обновляйте оба файла:
1. Добавьте новые разделы в структуру
2. Обновите метрики
3. Добавьте записи в "Недавние изменения"
