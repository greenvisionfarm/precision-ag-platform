# Список задач и улучшений

Этот файл содержит список задач, которые предстоит реализовать, а также известные проблемы и идеи для улучшения проекта.

## Текущие задачи

- [ ] Заменить нативные диалоги браузера (alert, prompt, confirm) на пользовательские, стилизованные диалоги (например, с использованием jQuery UI или легкой модальной библиотеки).
- [ ] Добавить возможность редактирования полей на карте (помимо удаления).
- [ ] Реализовать фильтрацию и поиск полей в таблице.
- [ ] Улучшить обработку ошибок загрузки shape-файлов (более детальные сообщения).
- [ ] Добавить аутентификацию и авторизацию пользователей.
- [ ] Разработать более продвинутый UI/UX для карты и боковой панели.
