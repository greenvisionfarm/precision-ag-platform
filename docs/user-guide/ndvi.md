# NDVI Анализ

NDVI (Normalized Difference Vegetation Index) — индекс вегетации, показывающий здоровье растений.

## Загрузка GeoTIFF

### Требования к файлу

- **Формат:** GeoTIFF (.tif, .tiff)
- **Размер:** До 1 GB
- **Проекция:** Любая (автоматическая конвертация)
- **Каналы:** Мультиспектральные (включая NIR и Red)

### Пошаговая инструкция

1. Откройте вкладку **📥 Загрузки**
2. В секции **Загрузка NDVI (GeoTIFF)** выберите файл
3. Нажмите **Загрузить**
4. Дождитесь обработки:
   - Загрузка файла (несколько секунд)
   - Анализ NDVI (фон, 10-30 секунд)
   - Векторизация зон
5. Автоматический переход на страницу поля

---

## Зонирование

### Автоматическое разбиение на зоны

Приложение автоматически создаёт 3 зоны продуктивности:

| Зона | Цвет | NDVI диапазон | Описание |
|------|------|---------------|----------|
| **Низкая** | 🔴 Красный | 0.0 - 0.3 | Угнетённые растения |
| **Средняя** | 🟡 Жёлтый | 0.3 - 0.6 | Нормальное состояние |
| **Высокая** | 🟢 Зелёный | 0.6 - 1.0 | Отличная вегетация |

### Алгоритм

1. Расчёт NDVI из мультиспектральных каналов
2. Гистограмма распределения значений
3. Кластеризация (k-means, k=3)
4. Векторизация полигонов
5. Сохранение в БД

## Визуализация

### Отображение зон на карте

1. Откройте **🗺️ Карта**
2. Выберите поле с NDVI
3. Включите слой **Зоны NDVI**
4. Кликните на зону для просмотра статистики

### Детальная информация

Для каждой зоны отображается:
- Площадь (га)
- Средний NDVI
- Процент от общей площади поля

---

## Использование для VRA

### Карта предписаний

На основе зон NDVI создаётся карта для дифференцированного внесения:

1. Откройте поле с NDVI
2. Нажмите **Создать карту предписаний**
3. Задайте нормы внесения (кг/га) для каждой зоны
4. Экспортируйте в Shapefile для техники

> **В разработке:** Редактор норм внесения

## Советы

### 🛰️ Съёмка

- Проводите съёмку в безоблачную погоду
- Оптимальное время: 10:00 - 14:00
- Высота полёта: 100-120 м для Mavic 3M

### 📊 Интерпретация

- **NDVI < 0.2:** Почва/отмершая растительность
- **NDVI 0.2-0.5:** Разреженная растительность
- **NDVI 0.5-0.8:** Здоровая вегетация
- **NDVI > 0.8:** Густая растительность

### ⚠️ Ограничения

- Облачность искажает результаты
- Тени от облаков создают артефакты
- Влажность почвы влияет на NDVI
