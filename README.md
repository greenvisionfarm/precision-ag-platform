# Field Mapper

Field Mapper — это веб-приложение для визуализации и управления географическими данными полей. Оно позволяет пользователям загружать shape-файлы в ZIP-архивах, отображать их на интерактивной карте, рисовать новые поля вручную, редактировать их границы и просматривать данные в виде таблицы.

## Основные возможности

- **Загрузка Shape-файлов:** Загрузка полей из ZIP-архива, содержащего shape-файл.
- **Рисование на карте:** Возможность вручную нарисовать новый полигон (поле) прямо на интерактивной карте.
- **Редактирование полей:** Возможность изменять границы существующих полей на карте с автоматическим пересчетом площади.
- **Интерактивная карта:** Отображение загруженных и созданных полей на карте с использованием Leaflet.
- **Визуализация данных:** Каждое поле можно нажать для просмотра основной информации (название, площадь).
- **Список полей:** Отдельная страница с таблицей всех полей, их площади и владельцев.
- **Удаление полей:** Возможность удалять поля как на карте, так и в таблице.
- **Переименование полей:** Возможность переименовывать поля как на карте, так и в таблице.
- **Управление владельцами:** Отдельная страница для добавления и просмотра владельцев.
- **Назначение владельца:** Возможность выбирать владельца поля прямо в таблице.

## Стек технологий

- **Backend:** Tornado (асинхронный), GeoPandas, Peewee (ORM), SQLite.
- **Frontend:** Leaflet.js, Leaflet.draw, DataTables, jQuery.

## API Эндпоинты

- `GET /`: Главная страница с картой.
- `GET /fields_list`: Страница с таблицей полей.
- `GET /owners`: Страница управления владельцами.
- `POST /upload`: Эндпоинт для загрузки ZIP-архива с shape-файлом.
- `GET /api/fields`: Возвращает все поля в формате GeoJSON.
- `GET /api/fields_data`: Возвращает данные для таблицы.
- `GET /api/owners`: Возвращает список всех владельцев.
- `POST /api/owner/add`: Добавляет нового владельца.
- `POST /api/field/add`: Добавляет новое поле вручную.
- `DELETE /api/field/delete/<id>`: Удаляет поле.
- `PUT /api/field/rename/<id>`: Переименовывает поле.
- `PUT /api/field/assign_owner/<id>`: Назначает владельца полю.
- `PUT /api/field/update_geometry/<id>`: Обновляет геометрию поля и пересчитывает его площадь (JSON: `{"geometry": {...}}`).

## Установка и запуск

1. `python -m venv venv`
2. `source venv/bin/activate`
3. `pip install -r requirements.txt`
4. `python app.py`
5. Откройте [http://localhost:8888](http://localhost:8888).
