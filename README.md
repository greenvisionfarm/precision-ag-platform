# Проект "Карта Полей"

Этот проект представляет собой веб-приложение для отображения географических полей на интерактивной карте. Бэкенд реализован на Python с использованием фреймворка Tornado, а фронтенд использует jQuery и библиотеку Leaflet.js для работы с картами. Данные о полях хранятся в базе данных SQLite.

## Структура проекта

-   `app.py`: Основной файл приложения Tornado. Определяет маршруты, обрабатывает HTTP-запросы, рендерит шаблоны и взаимодействует с базой данных.
-   `requirements.txt`: Список Python-зависимостей проекта (Tornado, Geopandas, Peewee).
-   `db.py`: Модуль для настройки базы данных SQLite и определения модели данных для полей (Field Model) с использованием ORM Peewee.
-   `templates/`: Директория для HTML-шаблонов.
    -   `index.html`: Главная HTML-страница, содержащая карту и форму для загрузки файлов.
-   `static/`: Директория для статических файлов (CSS, JavaScript, изображения).
    -   `js/`: Директория для JavaScript-файлов.
        -   `main.js`: JavaScript-код для инициализации карты Leaflet, взаимодействия с API бэкенда (загрузка данных, отправка файлов) и отрисовки полей.

## Функционал

1.  **Отображение полей:** Приложение загружает данные о полях из базы данных и отображает их на карте Leaflet.
2.  **Загрузка Shapefile:** Пользователь может загрузить ZIP-архив, содержащий Shapefile (набор файлов .shp, .dbf, .shx, .prj), через веб-форму. Бэкенд распаковывает архив, читает данные с помощью Geopandas и сохраняет их в базу данных.

## Запуск проекта

1.  Перейдите в корневую директорию проекта: `cd /home/vladibuyanov/PycharmProjects/field_mapper`
2.  Создайте и активируйте виртуальное окружение:
    ```bash
    python3 -m venv venv
    source venv/bin/activate
    ```
3.  Установите зависимости: `pip install -r requirements.txt`
    *   **Примечание:** Для `geopandas` могут потребоваться системные зависимости. Для Debian/Ubuntu: `sudo apt-get install -y libgdal-dev gdal-bin`
4.  Запустите сервер: `python app.py`
5.  Откройте в браузере: `http://localhost:8888`
